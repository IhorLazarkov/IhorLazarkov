    • Rental Residence Hub

      This Web App is platform to find residence for vacation. User can filter all available residences, read its details and reviews of others.

      Logged in user can CRUD (Create, Read, Update and Delete) owned properties and reviews. There is a demo user available to login to have access to the CRUD. A registered user can create own residence for rent with its latitude and longtitude to pin point its real location on a map. This functionality significanlty improves user experiance during search for an accomodation for next vacation.

      Wed app will determine user's geolocation and pre-populate 'country' filter criteria to provide all the properties within the country.

      Front-end: React, React-Router, HTML, CSS, Vite
      Backend-end: Express, Sequelize 6, SQLite, PostgrSQL

      Please be patient: the web app is hosted on free host-service therefor please be pasient for first load. I measured and it might take greater than 1min to receive a first bite.
